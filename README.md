# 推特视频下载软件

## 功能
- 下载推特视频（含音频）

## 使用方法

1. 安装依赖
```bash
pip install -r requirements.txt
```
如果没有安装 `ffmpeg`，请先安装 `ffmpeg`

2. 运行程序
```bash
python main.py
```

3. 输入推文URL
4. 下载完成

## 注意事项
- 推文URL需要是推特视频的URL，例如：https://x.com/dotey/status/1683738905412005888
